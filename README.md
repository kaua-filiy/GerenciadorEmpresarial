# Gerenciador Empresarial

Este é um projeto de sistema empresarial desenvolvido para fins de estudo, focado em consolidar meus conhecimentos em Programação Orientada a Objetos (POO), lógica de programação e no ecossistema C# com .NET 9. O projeto também visa aplicar boas práticas de versionamento utilizando Git e GitHub.

 O que foi estudado e aplicado?
Programação Orientada a Objetos (POO):
Encapsulamento: Implementação de proteção para dados sensíveis (ex: saldo e estoque), garantindo que as informações não sofram modificações indevidas fora das regras de negócio.

Abstração: Modelagem de entidades do mundo real, como Produto, Cliente e Vendas, transformando-as em classes organizadas.

Lógica de Negócio:
Desenvolvimento de algoritmos para controle de entrada e saída de estoque.

Cálculo automático de totais de venda.

Implementação de validação de dados para garantir a integridade das informações inseridas.

Boas Práticas:
Organização de código: Estruturação do projeto seguindo a responsabilidade de pastas: Models, Services e Repositories.

Ambiente Profissional: Configuração de um ambiente de desenvolvimento limpo, utilizando um arquivo .gitignore adequado para projetos .NET.

 Funcionalidades Inicias
 
Cadastro e gerenciamento de produtos.

Controle de entrada e saída de estoque.

 Registro de vendas com vinculação direta a clientes.

Como rodar o projeto

1-Clone o repositório:

git clone https://github.com/kaua-filiy/GerenciadorEmpresarial.git

2-Entre na pasta do projeto:

cd GerenciadorEmpresarial

3-Execute a aplicação:

dotnet run
